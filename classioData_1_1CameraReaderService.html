Detailed Description

Camera calibration file reader service.

Date:
2009.

Service reading camera calibration file and create a camera object.

  • This service has no specified start and stop method.
  • Use setConfiguration(cfg) and configure() methods to configure the filename.
  • Use update() to load the file, create the camera object and notify reading to observers.
