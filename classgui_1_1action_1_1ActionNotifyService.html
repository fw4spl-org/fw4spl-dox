void gui::action::ActionNotifyService::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

Configure the message to send.

Example of configuration : 

        <service ... >
            <notify type="::fwServices::ObjectMsg" event="APPLY" />
        </service>
  • <notify> :
    • type : the type of the message (i.e. fwServices::ObjectMsg)
    • event : the event of the message (i.e. "APPLY")
