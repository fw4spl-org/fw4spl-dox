Detailed Description

Mesh writer. Write file format .trian.

Date
2011

Ircad writer to write a fwData::Mesh (with only triangular cells) on filesystem which the file format is .trian. This format is basic. This is an ascii file which contains : 1) nb of points 2) list of points (x,y,z). Line numbers represent index of point in vector 3) nb of triangles 4) list of triangles with index of points and normals ( often set a -1 )
