void uiImage::WindowLevel::configuring ( ) throw (fwTools::Failed) [protected, virtual]

Configure the editor.

Example of configuration 

         <service uid="windowLevel" impl="::uiImage::WindowLevel" type="::gui::editor::IEditor" autoConnect="yes">
             <config autoWindowing="yes" selectedTFKey="mySelectedTF" tfSelectionFwID="myTFSelection" useImageGreyLevelTF="yes" />
         </service>

With :

  • autoWindowing : if 'yes', image windowing will be automatically compute from image pixel min/max intensity when this service receive BUFFER event
  • tfSelection : configure the identifier of the field containing the specific TF selection. By default, it use default selection field.
  • useImageGreyLevelTF : if 'yes' and if tfSelection is configured, then we use the grey level tf of image

Implements fwServices::IService.
