Detailed Description

IAdaptor implementation to show a cursor on histogram pointed by mouse.

xAxis : see scene2D::adaptor::IAdaptor

yAxis : see scene2D::adaptor::IAdaptor

zValue : see scene2D::adaptor::IAdaptor

color : inner color

borderColor : border color

opacity : from 0.0 to 1.0, default value is 0.8

viewportUID : a viewport that help us to manage the scaling of the graphic object
