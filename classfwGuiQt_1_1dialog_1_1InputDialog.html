Detailed Description

Defines an Input dialog.

Example of use:

inputDlg.setTitle("Identification dialog");
inputDlg.setMessage("Enter Pin Code: ");
inputDlg.setInput ("<Enter your code here>");
std::string inputText = inputDlg.getInput();
