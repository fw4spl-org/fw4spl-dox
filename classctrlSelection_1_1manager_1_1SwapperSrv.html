void ctrlSelection::manager::SwapperSrv::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

Implements configuring method derived from IService. .

Sample of declaration configuration for a simple swapper service

        <service uid="myManager" impl="::ctrlSelection::manager::SwapperSrv" type="::ctrlSelection::IManagerSrv" autoConnect="yes" >
            <mode type="dummy" />
            <config>
                <object id="myImage" type="::fwData::Image" >
                    <service uid="myMedicalImageConverter" impl="::ctrlSelection::MedicalImageSrv" type="::fwServices::IController"  autoConnect="no" />
                    <service uid="myServices" impl="..." type="..." autoConnect="yes" />

                    <connect>
                        <signal>key</signal>
                        <slot>uid/key</slot>
                    </connect>

                </object>
                <object id="myAcquisition" type="::fwData::Acquisition" >
                    <service uid="myServices2" impl="..." type="..." autoConnect="yes" worker="myThread" />

                    <proxy channel="...">
                        <signal>...</signal>
                        <signal>.../...</signal>
                        <slot>.../...</slot>
                        <slot>.../...</slot>
                    </proxy>
                </object>
            </config>
        </service>

With:

  • mode : must be "stop" or "dummy". The dummy mode doesn't stop the services when its attached object is deleted but swap it on a dummy object.
  • the objects and services tags are defined as same as the configuration of objects and services.
  • autoConnect: optional (default value = false), if true allows to listen signals from the associated object.
  • worker: optional, allows to manage the service in another thread.

Implements fwServices::IService.
