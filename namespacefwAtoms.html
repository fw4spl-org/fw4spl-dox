Detailed Description

fwAtoms contains basic objects to represent any other kind of object

fwAtoms contains seven basic types :

  • Numeric : real and integer type.
  • Boolean : boolean type.
  • String : String type.
  • Object : complex type.
  • Map : string:Atom association
  • Sequence : sequence of simple Atoms (Numeric, Boolean, String).
  • Blob: representing a raw buffer.
