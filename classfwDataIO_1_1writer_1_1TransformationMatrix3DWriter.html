Detailed Description

Writer for transformation of 3D Matrix. Write file format .trf.

Date
2009

Ircad writer to write a fwData::TransformationMatrix3D on filesystem which the file format is .trf. This format is basic. This is an ascii file which contains matrix 4x4 values
