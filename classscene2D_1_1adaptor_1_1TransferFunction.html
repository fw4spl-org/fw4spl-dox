void scene2D::adaptor::TransferFunction::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

Configuring the TransferFunction adaptor.

Example of configuration 

    <adaptor id="tf2" class="::scene2D::adaptor::TransferFunction" objectId="myImage">
        <config lineColor="lightGray" circleColor="lightGray" xAxis="xAxis" yAxis="yAxis" zValue="4"/>
    </adaptor>
  • <config lineColor="lightGray" circleColor="lightGray" xAxis="xAxis" yAxis="yAxis" zValue="4"/> : Set the config.

lineColor : no mandatory (default value : black) : Set the color of the lines between the TF points.

circleColor : no mandatory (default value : black) : Set the outline color of the circles representing the TF points.

xAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the x Axis of the TF layer.

yAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the y Axis of the TF layer.

zValue : no mandatory (default value : 0) : Set the zValue of the TF layer (the higher the zValue, the higher the layer is).

Reimplemented from scene2D::adaptor::IAdaptor.
