void visuVTKAdaptor::NegatoMPR::configuring ( ) throw (fwTools::Failed) [protected, virtual]

Configures the service. 

       <adaptor id="negato" class="::visuVTKAdaptor::NegatoMPR" objectId="imageKey">
           <config renderer="default" picker="negatodefault" mode="2d" slices="1" sliceIndex="axial"
                   transform="trf" tfalpha="yes" interpolation="off" vtkimagesource="imgSource" actorOpacity="1.0"
                   selectedTFKey="tkKey" tfSelectionFwID="selectionID" />
       </adaptor>
  • renderer (mandatory): defines the renderer to show the arrow. It must be different from the 3D objects renderer.
  • picker (mandatory): identifier of the picker
  • mode (optional, 2d or 3d): defines the scene mode. In 2d mode, the camera follow the negato in axial/frontal/sagital orientation. In 3d mode, the camera is automatically reset when the image is modified. If mode is not defined, the camera is free.
  • slices (optional, default=3): number of slices shown in the adaptor
  • sliceIndex (optional, axial/frontal/sagittal, default=axial): orientation of the negato
  • transform (optional): the vtkTransform to associate to the adaptor
  • tfalpha (optional, yes/no, default=no): if true, the opacity of the transfer function is used in the negato.
  • interpolation (optional, yes/no, default=yes): if true, the image pixels are interpolated
  • vtkimagesource (optional): source image, used for blend
  • actorOpacity (optional, default=1.0): actor opacity (float)
  • tfSelectionFwID (optional): fwID of the composite containing transfer functions
  • selectedTFKey (optional): key of the transfer function to use in negato
