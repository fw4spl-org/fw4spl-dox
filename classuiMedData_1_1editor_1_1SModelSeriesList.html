void uiMedData::editor::SModelSeriesList::configuring ( ) throw (fwTools::Failed) [protected, virtual]

Configures the editor.

Configuration example : 

     <enable_hide_all>true</enable_hide_all>
     <columns>
         <organ_name>@organ_name</organ_name>
         <volume_cc view="positive" >@volume</volume_cc>
     </columns>

enable_hide_all : if 'true', allows to hide all models through a single checkbox displayed in UI (default value is 'true', allowed values are 'true' and 'false').

columns : defines colums to be shown in reconstruction list. XML child element names follow fwData::Reconstruction serialization attribute names. The attribute 'view' is optional and can has the following values :

  • positive : a numeric value is displayed only if it is positive. Otherwise, 'Unknown' is displayed.
Exceptions:
fwTools::Failed
