00021 // .NAME fwVtkPicker - superclass for 3D geometric pickers (uses ray cast)
00022 // .SECTION Description
00023 // fwVtkPicker is used to select instances of vtkProp3D by shooting a ray
00024 // into a graphics window and intersecting with the actor's bounding box. 
00025 // The ray is defined from a point defined in window (or pixel) coordinates, 
00026 // and a point located from the camera's position.
00027 //
00028 // fwVtkPicker may return more than one vtkProp3D, since more than one bounding
00029 // box may be intersected. fwVtkPicker returns the list of props that were hit,
00030 // the pick coordinates in world and untransformed mapper space, and the 
00031 // prop (vtkProp3D) and mapper that are "closest" to the camera. The closest 
00032 // prop is the one whose center point (i.e., center of bounding box) 
00033 // projected on the ray is closest to the camera.
00034 
00035 // .SECTION See Also
00036 // fwVtkPicker is used for quick geometric picking. If you desire to pick
00037 // points or cells, use the subclass vtkPointPicker or vtkCellPicker,
00038 // respectively.  Or you may use hardware picking to pick any type of vtkProp
00039 // - see vtkPropPicker or vtkWorldPointPicker.
