Detailed Description

Base class for all services.

Date:
2007-2009.

This class defines the API to use and declare services. The service state aims at imposing method execution order (i.e. configure(), start(), update() or update(const fwServices::ObjectMsg::sptr), stop()).

Todo:

replace tests on status in start, stop, ... methods by assertions.

Refactoring of SWAPPING status. Perhaps must be a special status as UPDATING or UPDATING must be another GlobalStatus. it must be homogeneous.

Add a new method to test if m_associatedObject has expired
