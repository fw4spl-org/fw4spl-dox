Detailed Description

Render axis in the generic scene.

Slots

  • updateVisibility(bool) : show/hide the axis

XML Configuration

1 <service type="::visuVTKAdaptor::SAxis">
2  <config renderer="default" length="..." transform="..." label="..." marker="..." markerColor="..." />
3 </service>

Configuration

  • renderer(mandatory) : defines the renderer to show the axes.
  • transform(mandatory) : transformation matrix applied to the axes.
  • length(optional, default: 1.0) : defines the length of the axes.
  • label(optional, default: yes) : display the name of the axes.
  • marker(optional, default: no) : enable the sphere marker.
  • markerColor(optional, default: #FFFFFF) : color of the sphere marker.
