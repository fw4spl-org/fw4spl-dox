void visuVTKVRAdaptor::Volume::configuring ( ) throw (fwTools::Failed) [protected, virtual]

Configures the service.

Configuration example : 

     <adaptor id="VolumeScene3D" class="::visuVTKVRAdaptor::Volume" objectId="imageKey">
         <config renderer="default" clippingplanes="clippingPlanesId" autoresetcamera="yes|no" croppingBox="yes|no"/>
     </adaptor>

renderer : ID of renderer the adaptor must use clippingplanes : id of VTK object for clipping planes autoresetcamera : (not mandatory, default is yes) set if the renderer must reset camera when updating image croppingBox : (not mandatory, default is no) set if the cropping box must be shown

Exceptions:
fwTools::Failed

Implements fwServices::IService.
