Detailed Description

This service reads a dicom dir, and provides a fwMedData::SeriesDB.

Signals

  • jobCreated(SPTR(::fwJobs::IJob)): emitted to display a progress bar while the image is loading (it should be connected to a SJobBar).

Slots

  • readFolder(::boost::filesystem::path) : read the given file

XML Configuration

1 <service type="::ioVtkGdcm::SSeriesDBLazyReader">
2  <inout key="data" uid="..." />
3  <folder>...</folder>
4 </service>

In-Out

  • data [fwMedData::SeriesDB]: seriesDB that will contain the loaded image series and modelSeries.

Configuration

  • folder (optional): path of the folder containing the dicom files, if it is not defined, 'configureWithIHM()' should be called to define the path.
