void visuVTKAdaptor::ImageSliceOrientationText::configuring ( ) throw (fwTools::Failed) [protected, virtual]

Configure the adaptor.

Example : 

        <adaptor id="adaptorID" class="::visuVTKAdaptor::ImageSliceOrientationText" objectId="imageKey">
            <config renderer="default" >
                <locations>R,L,A,P,S,I</locations>
                <initialOrientation>${orientation}</initialOrientation>
            </config>
        </adaptor>
  • locations : comma-separated location terms to use, in this order : Right,Left,Anterior,Posterior,Superior,Inferior if 'default' is given, 'R,L,A,P,S,I' is used. If empty, nothing is displayed
  • initialOrientation : initial orientation of the associated slice

Implements fwServices::IService.
