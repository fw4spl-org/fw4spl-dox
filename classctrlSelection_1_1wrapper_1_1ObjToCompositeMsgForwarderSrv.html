void ctrlSelection::wrapper::ObjToCompositeMsgForwarderSrv::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

Configures the service. 

     <forward fromKey="objKey" onEvent="UPDATED_OBJECT" msgType="::fwComEd::ImageMsg" />

With this configuration : message with event "UPDATED_OBJECT" received from oject with key objKey" are forwarded to the composite. New message of type fwComEd::ImageMsg will be created and sent to the composite.

Note:
fromKey attibute can be "*", so the message can be received from any object.
