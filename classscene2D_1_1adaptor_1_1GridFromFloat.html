Configuring the GridFromFloat adaptor.

Example of configuration 

    <adaptor id="grid1" class="::scene2D::adaptor::GridFromFloat" objectId="myData">
        <config xMin="-750" xMax="750" yMin="-0.7" yMax="1.7" xSpacing="100" ySpacing="0.1" color="lightGray" xAxis="xAxis" yAxis="yAxis" zValue="1"/>
    </adaptor>
  • objectId : fwData::Float
  • <config xMin="-750" xMax="750" yMin="-0.7" yMax="1.7" xSpacing="100" ySpacing="0.1" color="lightGray" xAxis="xAxis" yAxis="yAxis" zValue="1"/> : Set the config.

xMin : mandatory : Set the minimum x value of the grid.

xMax : mandatory : Set the maximum x value of the grid.

yMin : mandatory : Set the minimum y value of the grid.

yMax : mandatory : Set the maximum y value of the grid.

xSpacing : no mandatory (default value : 10) : Set the grid spacing (space between 2 consecutive lines) in x.

ySpacing : no mandatory (default value : 10) : Set the grid spacing (space between 2 consecutive lines) in y.

color : no mandatory (default value : black) : Set the color of the lines.

xAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the x Axis of the grid layer.

yAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the y Axis of the grid layer.

zValue : no mandatory (default value : 0) : Set the zValue of the grid layer (the higher the zValue, the higher the layer is).
