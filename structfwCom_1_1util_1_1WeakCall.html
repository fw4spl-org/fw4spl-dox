Detailed Description

template<typename T, typename R>
struct fwCom::util::WeakCall< T, R >

Weak functor class. Store a zero-arg-function and a weak pointer on an object. The WeakCall is callable only once. A bad_weak_ptr exception is thrown if the weak_ptr is expired when operator() is called.
