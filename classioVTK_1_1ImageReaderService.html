io::IOPathType ioVTK::ImageReaderService::getIOPathType ( ) const [protected, virtual]

This method must be implemented by concrete service readers that use path file system to read data.

This method returns supported path format (file, files or folder). A reader can support file and folder, or files and folder, but not file and files ( because files include file concept ).

Reimplemented from io::IReader.
