Detailed Description

Transformation matrix 3D writer service.

Date:
2009.

Service writing a TransformationMatrix3D object

  • Use setConfiguration(cfg) and configure() methods to configure the matrix filename.
  • Use start() to initialize the service.
  • Use update() to write the transformation matrix.
  • Use stop() to stop service before to destroy it.
