Detailed Description

Adaptor implementation that manages the camera on the view (by updating the viewport object).

The ViewportUpdater updates the viewport of the scene it is attached to, by picking values from the managed scene2D::data::Viewport object.

Configuration example: 

   <adaptor id="viewportRangeUpdater" class="::scene2D::adaptor::ViewportUpdater" objectId="myViewport">
       <config xAxis="xAxis" yAxis="yAxis" zValue="7" />
   </adaptor>

objectID : the viewport object this updater takes values from.

xAxis : see scene2D::adaptor::IAdaptor

yAxis : see scene2D::adaptor::IAdaptor

zValue : see scene2D::adaptor::IAdaptor

This adaptor is intended to be used with a scene2D::adaptor::ViewportRangeSelector adaptor.
