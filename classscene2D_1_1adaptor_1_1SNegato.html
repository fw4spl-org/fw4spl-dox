Detailed Description

IAdaptor implementation to display one slice of an image.

Slots

  • updateSliceIndex() : update image slice index
  • updateSliceType() : update image slice type
  • updateBuffer() : update image buffer
  • updateVisibility() : update image visibility
  • updateTFPoints() : update the displayed image according to the new points
  • updateTFWindowing(double window, double level) : update the displayed image according to the new window and level

XML Configuration

1 <service uid="negato" type="::scene2D::adaptor::SNegato" autoConnect="yes">
2  <inout key="image" uid="..." />
3  <inout key="tf" uid="..." />
4  <config xAxis="xAxis" yAxis="yAxis" color="gray" opacity="0.25" zValue="5" />
5 </service>

In-Out

  • image [fwData::Image]: image to display.
  • tf [fwData::TransferFunction] (optional): the current TransferFunction. If it is not defined, we use the image's default transferFunction (CT-GreyLevel). The transferFunction's signals are automatically connected to the slots 'updateTFPoints' and 'updateTFWindowing'.

Configuration

  • config (mandatory): contains the adaptor configuration
    • xAxis (optional): x axis associated to the adaptor
    • yAxis (optional): y axis associated to the adaptor
    • zValue (optional, default=0): z value of the layer
    • opacity (optional, default=1.0): adaptor opacity (float)
    • orientation (optional, default axial): image orientation, axial, sagittal or frontal
    • changeSliceType (optional, default true): specify if the negato allow slice type events
