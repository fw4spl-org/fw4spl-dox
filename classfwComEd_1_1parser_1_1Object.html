Detailed Description

Specific fwTools::Object parser. Does nothing as no IService method is specialized: IXMLParser base class methods are therefore considered.

Note:
This implementation is only written to avoid the mixing both the service type IXMLParser and the fwTools::Object implementation (for homogeneity).
