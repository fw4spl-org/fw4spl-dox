Detailed Description

This service is used to pull series from a PACS.

Signals

  • progressed(std::string) : Signal to start the progress (bar id).
  • startedProgress(std::string, float, std::string) : Signal to update the progress (bar id, percentage, message).
  • stoppedProgress(std::string) : Signal to stop the progress (bar id).

XML Configuration

1  <service type="::namespace::classname">
2  <in key="pacsConfig" uid="..." />
3  <in key="selectedSeries" uid="..." />
4  <inout key="seriesDB" uid="..." />
5  <config dicomReader="::ioGdcm::SSeriesDBReader" dicomReaderConfig="config" />
6 </service>

Input

  • pacsConfig [fwPacsIO::data::PacsConfiguration]: PACS configuration data.
  • selectedSeries [fwData::Vector]: List of DICOM series to pull from the PACS..

In-Out

  • seriesDB [fwMedData::SeriesDB]: SeriesDB where to put the retrieved dicom series.

Configuration

  • dicomReader Reader type to use.
  • dicomReaderConfig Optional configuration for the DICOM Reader.
