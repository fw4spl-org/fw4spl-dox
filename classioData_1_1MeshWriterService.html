Detailed Description

Mesh writer service.

Date:
2011.

Service writing a mesh object.

  • This service has no specified start and stop method.
  • Use setConfiguration(cfg) and configure() methods to configure the mesh filename.
  • Use update() to write the mesh and notify observers.
