Detailed Description

Cursor management: displays a red cross representing the image picked point.

Slots

  • updateSliceIndex(int axial, int frontal, int sagittal) : update image slice index

XML Configuration

1 <service type="::visuVTKAdaptor::SProbeCursor" autoConnect="yes">
2  <inout key="image" uid="..." />
3  <config renderer="default" picker="negatodefault" transform="trf" />
4 </service>

In-Out

  • image [fwData::Image]: image to display.

Configuration

  • config(mandatory) : contains the adaptor configuration
    • renderer (mandatory): defines the renderer to show the SProbeCursor.
    • picker (mandatory): identifier of the picker
    • transform (optional): the vtkTransform to associate to the adaptor
