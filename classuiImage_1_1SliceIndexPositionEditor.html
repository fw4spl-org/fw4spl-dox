Detailed Description

SliceIndexPositionEditor service allows to change the slice index of an image.

This is represented by

  • a slider to select the slice index
  • a choice list to select the slice orientation (axial, frontal, sagittal)

XML Configuration

1 <service uid="..." type="::uiImage::SliceIndexPositionEditor" autoConnect="yes">
2  <inout key="image" uid="..."/>
3  <sliceIndex>${orientationValue}</sliceIndex>
4 </service>

In-Out

  • image [fwData::Image]: image on which the slice index will be changed

Configuration

  • sliceIndex : Axis on which the index will be changed, must be "axial", "frontal" or "sagittal".
