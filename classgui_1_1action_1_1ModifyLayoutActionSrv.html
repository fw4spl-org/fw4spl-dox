void gui::action::ModifyLayoutActionSrv::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

This method is used to configure the service parameters: specifies which views to show/hide/mode. 

     <config>
         <move uid="srv_uid" wid="new_view_wid" />
         <show wid="view_to_show" />
         <hide wid="view_to_hide />
         <show_or_hide wid="view_to_show_or_hide />
         <enable uid="action_to_enabled" />
         <disable uid="action_to_disabled" />
     </config>
  • move configures the services to move in the new view given by wid
  • show configures the view to show
  • hide configures the view to hide
  • show or hide configures the view to show or hide. If the action is activated, the view is shown else it is hidden. The action in the menu/toolbar must be checkable (style="check").
