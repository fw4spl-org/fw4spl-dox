void scene2D::adaptor::Line::configuring ( ) throw ( ::fwTools::Failed ) [protected, virtual]

Configuring the Line adaptor.

Example of configuration 

    <adaptor id="abscissa" class="::scene2D::adaptor::Line" objectId="self">
        <config x1="-500" x2="500" y1="0" y2="0" color="white" xAxis="xAxis" yAxis="yAxis" zValue="2" />
    </adaptor>
  • <config x1="-500" x2="500" y1="0" y2="0" color="white" xAxis="xAxis" yAxis="yAxis" zValue="2" /> : Set the config.

x1 : mandatory : Set the line begining coordinate x value.

x2 : mandatory : Set the line ending coordinate x value.

y1 : mandatory : Set the line begining coordinate y value.

y2 : mandatory : Set the line ending coordinate y value.

color : no mandatory (default value : black) : Set the color of the line.

xAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the x Axis of the line layer.

yAxis : no mandatory (default value : ::scene2D::data::Axis::New() : m_origin (0), m_scale (1), m_scaleType (LINEAR)) : Set the y Axis of the line layer.

zValue : no mandatory (default value : 0) : Set the zValue of the line layer (the higher the zValue, the higher the layer is).
