void fwGui::layoutManager::IMenuBarLayoutManager::initialize ( ConfigurationType  configuration) [virtual]

Initialize layout managers.

Example of configuration 

      <service uid="menuBar" type="::fwGui::IMenuBarSrv" impl="::gui::aspect::DefaultMenuBarSrv" autoConnect="no" >
          <gui>
              <layout>
                  <menu name="My Menu"/>
                  <menu name="My Menu 2"/>
              </layout>
          </gui>
          <registry>
              <menu sid="myMenu" start="yes" />
              <menu sid="myMenu2" start="yes" />
          </registry>
      </service>

This method analyzes the gui section of the configuration.

  • <layout> (mandatory) : give the list of the menu that will appear in the menu bar.
  • <menu name="My Menu"/> :
    • name (mandatory) : give the name of the menu that will appear in the interface.
