void fwGui::IActionSrv::initialize ( ) [protected]

Initialize the action.

Example of configuration 

           <service uid="item" type="::fwGui::IActionSrv" impl="::gui::action::NothingActionSrv" autoConnect="no" >
              <state active="false" executable="false" inverse="true" />
              <confirmation value="yes" message="..." />
           </service>
  • <state active="false" executable="false" /> : fix the state of the action in the menu and Toolbar.
    • executable not mandatory (Default value true ) : allows to enable/disable the execution of the action. If the action appears in the interface it will be enable/disable.
    • active not mandatory (Default value false ): If the action appears in the interface it will be check/uncheck.
    • inverse not mandatory (Default value true) : allow to invert the state of the action (if "check")
  • <confirmation value="yes" message="..." /> : configure if the action must be confirm by user before to execute it.
    • value : if true the action will show a confirmation message before to be executed.
    • message not mandatory : if not empty the message is shown in dialog box.
    • defaultbutton not mandatory (default defined by underlying gui backend) : specify the dialog default button (yes or no)
