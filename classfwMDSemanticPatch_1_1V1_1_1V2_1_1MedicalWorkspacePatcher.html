Detailed Description

This patcher adds required meta infos on composites describing a 'Medical workspace'.

Composite describing a 'Medical workspace' contains childs designated by keys 'patientDB', 'planningDB' and 'processingDB'.

Applies the patch to the specified object.

Adds meta info 'compositeType' with value 'MedicalWorkspace' if current object has childs designated with keys 'patientDB', 'planningDB' and 'processingDB', and applies patcher 'DefaultPatcher'.
