Detailed Description

Defines the service interface managing the rendering service for object.

Date:
2007-2009.

Use the base service methods :

  • The service is configured with methods setConfiguration(cfg) and configure()
  • The method start() initialize the service
  • Use update() method to update data
  • To react on modifications use update(msg)
  • Finally we must call stop() before deleting the service
