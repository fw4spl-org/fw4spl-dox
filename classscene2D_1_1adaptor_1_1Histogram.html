Detailed Description

IAdaptor implementation for histogram data.

Configuration example:

   <adaptor id="histogram" class="::scene2D::adaptor::Histogram" objectId="myHistogram">
       <config xAxis="xAxis" yAxis="axeHistogramY" color="gray" opacity="0.25" zValue="5"/>
   </adaptor>

color (mandatory) : the background color of the histogram

opacity (mandatory) : the opacity of the histogram (from 0.0 to 1.0)

xAxis : see scene2D::adaptor::IAdaptor

yAxis : see scene2D::adaptor::IAdaptor

zValue : see ::scene1D::adaptor::IAdaptor
